About Me

I'm Joscha, I develop software and hardware tools for scientific research. Currently, I work for the Department of Biology and Chemistry at University of Bremen, Germany. In addition, I'm available as a freelance software developer.

Technologies

I have experience working with a wide range of technologies, from C++ and custom electronics for interactive audio-visual stimulation to Python and MATLAB for data pipelines and high-performance computing. I have also worked with web and database systems to create comprehensive solutions for scientific research.

Projects

I have worked on a number of projects of varying scope and different levels of involvement. Mostly, I worked in the fields of experimental control, data acquisition, analysis, and management. Here are some of the (public) projects I have worked on:

VStim
Comprehensive 2D engine for interactive, audio-visual behavioral experiments, written in C++ using DirectX and MFC
neuro-conda
A curated Python distribution for Neuroscience based on conda
ARCADE
ARCADE is a modular suite of Applications for Real-time Control of subject Actions and stimulus Display, written in C++ and MATLAB
Metadata Editor for Research Data
A Nextcloud app to annotate research data, written in PHP and JavaScript using React and JSONforms
Syncopy
Python package for Systems Neuroscience Computing in Python aiming to bring easily scalable parallelization to data Neuroscience analysis
DomeVR
Immersive dome projection for naturalistic behavioral experiments, written in C++ using Unreal Engine 4
www.esi-frankfurt.de
Website development using HTML, CSS and JavaScript with Hugo
